Overview:

Javascript Wikipedia

Javascript is a high level programming language that was written in 10 days.
Since then it has been added to and standardized by the E.C.M.A.(European Computer Manufacturers Association)
Javascript is different from the Java Programming Language but was influenced by its popularity.
Javascript is the default programming language of the World Wide Web A.K.A. The Internet.
This makes javascript (in my opinion) the most important programming language in the world.
The top 10 sites with the most traffic use Javascript.
They might have varying back-end or server side programming languages but, Javascript is neccesary for dynamic web pages.
For simplicity this page will focus on Vanilla or Basic Javascript and not Frameworks/Libraries.
